## Getting Started

**To get this project set up on your local machine, follow these steps:**

1. Open Terminal.

2. Navigate to your desired location to download the contents of this repository.

3. Run  
```
git clone git@github.com:Tchilo/react-tailwind.git
```
4. Run
  ```
  cd react-tailwind.git
  ```
5. Run 
```
npm install` or `npm i
```
6. Run 
```
npm run start
``` 
7. start building!

8. The http://localhost:1234/ server will automatically open in your browser, if it doesn't hold down the ctrl key then click the link in your terminal to open it in your browser. happy coding!! ğŸŒˆ
